# SwiftCloud Readme

## Why [ClickHouse](https://clickhouse.com/)?

This project utilizes ClickHouse for its data storage and querying needs. ClickHouse is a high-performance, open-source columnar database that excels at handling large volumes of data and executing complex analytical queries with low latency. Its ability to efficiently compress and index data makes it an ideal choice for applications that require fast access to large datasets.

## API Documentation

The Postman can be found in the `SWFIT CLOUD.postman_collection.json` file of the root directory. Please refer to the [API Docs](http://localhost:3000/api-docs/) **Swagger Docs** for detailed information on available endpoints, request/response formats, and usage examples.

## How to Run the Project

Follow these steps to set up and run the project:

1. **Prerequisites**

    - Ensure you have [NodeJS16](https://nodejs.org/download/release/v16.20.2/) installed.
    - Ensure you have [Docker](https://www.docker.com/products/docker-desktop/) installed

2. **Running the Application**

    - Execute the following command to start the application:

        ```bash
         npm run shake-it-off
        ```

    - The application should now be running and accessible at [http://localhost:3000](http://localhost:3000).
